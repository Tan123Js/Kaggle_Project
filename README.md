# Data Analytics Portfolio

## Projects

1. Retail Sales Analysis  
2. Customer Churn Analysis  
3. Brazilian E-Commerce Analysis

Each project includes:

- Exploratory Data Analysis (EDA)
- Data visualization
- Business insights and recommendations
